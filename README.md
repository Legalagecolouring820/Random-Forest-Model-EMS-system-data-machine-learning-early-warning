## 🔧 Basic Troubleshooting

### The app does not open

- Check that the file finished downloading
- Try running it as an administrator
- Move the file to your Desktop and open it again
- Make sure Windows did not block the file

### The app says it cannot find data

- Check the file path
- Make sure the file still exists
- Confirm the file type is correct
- Move the data file to a simple folder path like `C:\EMS\Data`

### The app shows no warning

- Make sure you loaded records first
- Check that the model file is selected
- Confirm the data has enough entries
- Run the check again after updating the source file

### The update stays paused

- Look for missing maintenance fields
- Complete the record
- Run the validation step again
- Refresh the app view
